Miapi-node

Mi interpretación de la API de Node v0.8.8

Aserciones

Estabilidad: 5 — Cerrada

Este módulo se usa para escribir pruebas unitarias para tus aplicaciones, puedes acceder a él con:

var acierta = require('assert');

acierta.fail(real, esperado, mensaje, operador)

Lanza una excepción que exhibe los valores para real y esperado separados por el operador proporcionado.

acierta(valor, mensaje), acierta.ok(valor, [mensaje])

Prueba si el valor es true, esta es equivalente a acierta.equal(true, !!valor, mensaje);

acierta.equal(real, esperado, [mensaje])

Prueba superficial, de igualdad coactiva con el operador de comparación igual ( == ).

acierta.notEqual(real, esperado, [mensaje])

Prueba superficial, de desigualdad coactiva con el operador de comparación de desigualdad ( != ).

acierta.deepEqual(real, esperado, [mensaje])

Prueba profunda de igualdad.

acierta.notDeepEqual(real, esperado, [mensaje])

Prueba profunda de cualquier desigualdad.

acierta.strictEqual(real, esperado, [mensaje])

Prueba la igualdad estricta, como lo determina el operador de igualdad estricta ( === )

acierta.notStrictEqual(real, esperado, [mensaje])

Prueba la desigualdad estricta, como lo determina el operador de desigualdad estricta ( !== )

acierta.throws(bloque, [error], [mensaje])

Espera bloque para lanzar un error. El error puede ser un constructor, una expresión regular o una función de validación.

Valida instanceof usando el constructor:

assert.throws(
  function () {
    throw new Error('Valor incorrecto');
  },
  Error
);

Valida el mensaje de error usando la expresión regular, RegExp:

assert.throws(
  function () {
    throw new Error('Valor incorrecto');
  },
  /valor/
);

Error de validación personalizado:

assert.throws(
  function () {
    throw new Error('Valor incorrecto');
  },
  function (error) {
    if ( (error instanceof Error) && /valor/.test(error) ) {
      return true;
    }
  },
  'error imprevisto'
);

acierta.doesNotThrow(bloque, [error], [mensaje])

Espera bloque para no lanzar un error, ve acierta.throws para detalles.

acierta.ifError(valor)

Prueba si valor no es un valor false, lanzado si valor es true. Útil cuando pruebas el primer argumento, error en retrollamadas.

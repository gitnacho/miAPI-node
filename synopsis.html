Miapi-node

Mi interpretación de la API de Node v0.8.8

Sinopsis

Un ejemplo de un servidor web, escrito en Node que responde con un 'Hola mundo':

var http = require('http');

http.createServer(function (peticion, respuesta) {
  respuesta.writeHead(200, {'Content-Type': 'text/plain'});
  respuesta.end('Hola, mundo

'); }).listen(8124);

console.log('Servidor escuchando en http://127.0.0.1:8124/');

Para ejecutar el servidor, guarda el código en un archivo llamado holamundo.js y ejecútalo con el intérprete de Node:

$ node holamundo.js
Servidor escuchando en http://127.0.0.1:8124/

Todos los ejemplos en la documentación se pueden ejecutar de manera similar.

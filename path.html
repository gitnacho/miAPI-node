Ruta

Estabilidad: 3 — Estable

Este módulo contiene utilidades para manipular y transformar rutas de archivo. Casi todos estos métodos sólo realizan transformación de cadenas. No se consulta al sistema de archivos para comprobar si las rutas son válidas.

Para utilizar este módulo usa:

var ruta = require('path'); // para usarla en
                            // los ejemplos

Proporciona los siguientes métodos:

ruta.normalize(ruta)

Normaliza una cadena de ruta, teniendo cuidado de las partes '..' y '.'.

Cuando se encuentran varias barras inclinadas, son reemplazadas por una sola; Cuando la ruta contiene una barra inversa, se conserva. Windows utiliza barras inversas.

Ejemplo:

ruta.normalize('/loquesea/bar//baz/asdf/quux/..')
// devuelve
'/loquesea/bar/baz/asdf'

ruta.join([ruta1], [ruta2], [...])

Une todos los argumentos y normaliza la ruta producida. Los argumentos distintos de cadena se omiten.

Ejemplo:

ruta.join('/loquesea', 'bar', 'baz/asdf', 'quux', '..')
// devuelve
'/loquesea/bar/baz/asdf'

ruta.join('loquesea', {}, 'bar')
// devuelve
'loquesea/bar'

ruta.resolve([desde...],hasta)

Resuelve hasta en una ruta absoluta.

Si hasta no es absoluta ya, los argumentos desde se anteponen en orden de derecha a izquierda, hasta que encuentre una ruta absoluta. Si después de utilizar todas las rutas desde todavía no encuentra ninguna ruta absoluta, utiliza el directorio de trabajo actual también. La ruta producida se normaliza y se quitan las barras finales, a menos que la ruta obtenida resuelva al directorio raíz. Los argumentos distintos de cadena se omiten.

Otra forma de pensar es como una secuencia de ordenes cd en un intérprete.

ruta.resolve('loquesea/bar', '/tmp/archivo/', '..', 'a/../subarchivo')

Es similar a:

cd loquesea/bar
cd /tmp/archivo/
cd ..
cd a/../subarchivo
pwd

La diferencia es que no es necesario que existan las diferentes rutas y también pueden ser archivos.

Ejemplos:

ruta.resolve('/loquesea/bar', './baz')
// devuelve
'/loquesea/bar/baz'

ruta.resolve('/loquesea/bar', '/tmp/archivo/')
// devuelve
'/tmp/archivo'

ruta.resolve('wwwroot', 'static_files/png/', '../gif/image.gif')
// si actualmente estás en /home/yo/node, devolverá
'/home/yo/node/wwwroot/static_files/gif/image.gif'

ruta.relative(desde, hasta)

Resuelve la ruta de acceso relativa desde a hasta.

A veces tenemos dos rutas absolutas y tenemos que derivar la ruta relativa de una a otra. Esta realmente transforma la inversa de ruta.resolve, lo cual significa que vemos:

ruta.resolve(desde, ruta.relative(desde, hasta)) == ruta.resolve(hasta)

Ejemplos:

ruta.relative('C:\orandea\prueba\aaa', 'C:\orandea\impl\bbb')
// devuelve
'..\..\impl\bbb'

ruta.relative('/data/orandea/prueba/aaa', '/data/orandea/impl/bbb')
// devuelve
'../../impl/bbb'

ruta.dirname(ruta)

Devuelve el nombre del directorio de la ruta. Similar a la orden dirname de Unix.

Ejemplo:

ruta.dirname('/loquesea/bar/baz/asdf/quux')
// devuelve
'/loquesea/bar/baz/asdf'

ruta.basename(ruta, [extension])

Devuelve la última parte de una ruta. Similar a la orden basename de Unix.

Ejemplo:

ruta.basename('/loquesea/bar/baz/asdf/quux.html')
// devuelve
'quux.html'

ruta.basename('/loquesea/bar/baz/asdf/quux.html', '.html')
// devuelve
'quux'

ruta.extname(ruta)

Devuelve la extensión de la ruta, desde el último '.' hasta el final de la cadena en la última parte de la ruta. Si no hay un '.' en la última parte de la ruta o el primer carácter de la misma es '.', entonces, devuelve una cadena vacía. Ejemplos:

ruta.extname('index.html')
// devuelve
'.html'

ruta.extname('index.')
// devuelve
'.'

ruta.extname('index')
// devuelve
''

ruta.sep

El separador de archivo específico del ambiente. '\' o '/'.

Un ejemplo en linux:

'loquesea/bar/baz'.split(ruta.sep)
// devuelve
['loquesea', 'bar', 'baz']

Un ejemplo de windows:

'loquesea\bar\baz'.split(ruta.sep)
// devuelve
['loquesea', 'bar', 'baz']

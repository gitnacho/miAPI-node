Miapi-node

Mi interpretación de la API de Node v0.8.8

Sistema operativo

Estabilidad: 4 — API congelada

Proporciona algunas funciones de utilidad básica relacionadas con el sistema operativo.

Para acceder a este módulo usa:

var os = require('os');

os.tmpDir()

Devuelve el directorio predeterminado del sistema operativo para los archivos temporales.

os.hostname()

Devuelve el nombre del servidor en el sistema operativo.

os.type()

Devuelve el nombre del sistema operativo.

os.platform()

Devuelve el ambiente del sistema operativo.

os.arch()

Devuelve la arquitectura del CPU en el sistema operativo.

os.release()

Devuelve la versión del sistema operativo.

os.uptime()

Devuelve la cantidad de tiempo en segundos que ha estado activo el sistema.

os.loadavg()

Devuelve un arreglo que contiene los promedios de carga de 1, 5 y 15 minutos.

os.totalmem()

Devuelve la cantidad total de memoria en bytes del sistema.

os.freemem()

Devuelve la cantidad de memoria libre del sistema, en bytes.

os.cpus()

Devuelve un arreglo de objetos que contienen información acerca de cada CPU/núcleo instalado: modelo, velocidad (en MHz) y tiempos (un objeto que contiene el número de pulsos del CPU pasado en: user, nice, sys, idle e irq).

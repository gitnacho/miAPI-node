Cifrado débil con «punycode»

Estabilidad: 2 — Inestable

Punycode.js viene con Node.js v0.6.2+. Para acceder a este módulo usa:

var punycode = require('punycode'); // Este módulo hace un
                                    // cifrado/decifrado débil

(Para usarlo con otras versiones de Node.js, primero utiliza npm para instalar el módulo punycode).

punycode.decode(cadena)

Convierte una cadena Punycode en puntos de codificación ASCCI a puntos de codificación Unicode.

// decodifica las partes del nombre de dominio
punycode.decode('maana-pta'); // 'mañana'
punycode.decode('--dqo34k'); // '☃-⌘'

punycode.encode(cadena)

Convierte una cadena en puntos de codificación Unicode a una cadena Punycode en puntos de codificación ASCII.

// codifica partes del nombre de dominio
punycode.encode('mañana'); // 'maana-pta'
punycode.encode('☃-⌘'); // '--dqo34k'

punycode.toUnicode(dominio)

Convierte una cadena Punycode que representa un nombre de dominio a Unicode. Sólo se convertirán las partes del nombre de dominio codificadas débilmente, es decir, no importa si la llamas con una cadena que ya se ha convertido a Unicode.

// decodifica nombres de dominio
punycode.toUnicode('xn--maana-pta.com'); // 'mañana.com'
punycode.toUnicode('xn----dqo34k.com'); // '☃-⌘.com'

punycode.toASCII(dominio)

Convierte una cadena Unicode que representa un nombre de dominio a Punycode. Sólo se convertirán las piezas no ASCII del nombre de dominio, es decir, no importa si la llamas con un dominio que ya está en ASCII.

// codifica nombres de dominio
punycode.toASCII('mañana.com'); // 'xn--maana-pta.com'
punycode.toASCII('☃-⌘.com'); // 'xn----dqo34k.com'

punycode.ucs2

punycode.ucs2.decode(cadena)

Crea un arreglo conteniendo los puntos de codificación decimal de cada carácter Unicode en la cadena. Si bien JavaScript internamente utiliza UCS-2, esta función convertirá las mitades en un par suplementario (que UCS-2 expone como diferentes caracteres) en un único punto de código, coincidiendo con UTF-16.

punycode.ucs2.decode('abc'); // [97, 98, 99]
// par suplementario del tetragrama U+1D306 para centrar:
punycode.ucs2.decode('\uD834\uDF06'); // [0x1D306]

punycode.ucs2.encode(puntosDeCodificacion)

Crea una cadena pasada en un arreglo en puntos de codificación decimal.

punycode.ucs2.encode([97, 98, 99]); // 'abc'
punycode.ucs2.encode([0x1D306]); // '\uD834\uDF06'

punycode.version

Una cadena que representa el número de versión actual de Punycode.js.

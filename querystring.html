Cadena de consulta

Estabilidad: 3 — Estable

Este módulo proporciona utilidades para procesar cadenas de consulta. Para acceder a él usa:

var cadenaDeConsulta = require('querystring');

Este provee los siguientes métodos:

cadenaDeConsulta.stringify(objeto, [separador], [asignacion])

Serializa un objeto a una cadena de consulta. Opcionalmente sustituye los caracteres predefinidos del separador ('&') y de asignacion ('=').

Ejemplo:

cadenaDeConsulta.stringify({ loquesea: 'bar',
  baz: ['qux', 'quux'], corge: ''
})
// regresa
'loquesea=bar&baz=qux&baz=quux&corge='

cadenaDeConsulta.stringify({loquesea: 'bar',
    baz: 'qux'}, ';', ':')
// regresa
'loquesea:bar;baz:qux'

cadenaDeConsulta.parse(cadena, [separador], [asignacion], [opciones])

Deserializa una cadena de consulta a un objeto. Opcionalmente sustituye los caracteres predefinidos del separador ('&') y de asignacion ('=').

Las opciones del objeto pueden contener la propiedad maxKeys (igual a 1000 de manera predeterminada), la cual utilizará para limitar las claves procesadas. Ponla a 0 para eliminar la limitación de claves.

Ejemplo:

cadenaDeConsulta.parse('loquesea=bar&baz=qux&baz=quux&corge')
// regresa
{ loquesea: 'bar',
  baz: ['qux', 'quux'], corge: '' }

cadenaDeConsulta.escape

La función de escape utilizada por cadenaDeConsulta.stringify, proporcionada a modo de que se pueda anular de ser necesario.

cadenaDeConsulta.unescape

La función unescape utilizada por cadenaDeConsulta.parse, proporcionada a modo de que se pueda anular de ser necesario.

DecodificadorDeCadena

Estabilidad: 3 — Estable

Para utilizar este módulo, usa:

var decodificadorDeCadena = require('string_decoder');

DecodificadorDeCadena decodifica un búfer a una cadena. Se trata de una sencilla interfaz para buffer.toString(), pero ofrece compatibilidad adicional para UTF-8.

var decodificadorDeCadena = require('string_decoder').StringDecoder;
var decodificadorDeCadena = new StringDecoder('utf8');

var centavo = new Buffer([0xC2, 0xA2]);
console.log(decodificadorDeCadena.write(centavo));

var euro = new Buffer([0xE2, 0x82, 0xAC]);
console.log(decodificadorDeCadena.write(euro));

Clase: StringDecoder

Acepta un sólo argumento, codificacion el cual de manera predeterminada es utf8.

DecodificadorDeCadena.write(bufer)

Devuelve una cadena decodificada.

Miapi-node

Mi interpretación de la API de Node v0.8.8

console

Estabilidad: 4 — API congelada
  • {Objeto}

Para imprimir en la salida y error estándar ('stdout' y 'stderr', respectivamente). Similar a las funciones del objeto console proporcionado por la mayoría de los navegadores web, aquí la salida se envía a 'stdout' o a 'stderr'.

console.log([datos], [...])

Imprime en la salida estándar con salto de línea. Esta función puede tomar varios argumentos de manera similar a la forma en que los maneja printf(). Ejemplo:

console.log('cantidad: %d', cantidad);

Si los elementos de formato no se encuentran en la primera cadena se utiliza util.inspect en cada argumento. Consulta util.format() para más información.

console.info([datos], [...])

Igual que console.log.

console.error([datos], [...])

Igual que console.log pero imprime a la salida de error estándar —stderr.

console.warn([datos], [...])

Igual que console.error.

console.dir(objeto)

Usa util.inspect en el objeto objeto e imprime la cadena resultante a la salida estándar.

console.time(etiqueta)

Marca un temporizador.

console.timeEnd(etiqueta)

Finaliza el temporizador, anotando la salida. Ejemplo:

console.time('100-elementos');
for (var i = 0; i < 100; i++) {
  ;
}
console.timeEnd('100-elementos');

console.trace(etiqueta)

Imprime el seguimiento de la pila hasta la posición actual en la salida de error estándar —'stderr'.

console.assert(expresion, [mensaje])

Igual que assert.ok() dónde si la expresion evalúa como false lanza un AssertionError con mensaje.
